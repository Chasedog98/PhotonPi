Gcode Terminal PhotonPi V1.1.0

G0 Z10

Move Z axis at max speed

G0 Z10 F600

Move Z axis at F speed

G28 Z

Home Z Axis

G90

Absolute Positioning

G91

Incremental Positioning

M20

List SD Card

M23

Select File on SD Card

M24

Starts printing the M6032 open file on memory

M25

Pause Print

M27

Report SD print status

M30

Delete file from SD Card

M33

Stops print *

M105

returns sensor temperatures

M106

Turn on Fan, only works during print

M107

Turn off Fan, Only works during print

M114

Get current position

M115

Firmware Info

M4001

Returns the configured stepper movement for all axis

M6030"*"

starts a print file name imeddiatly

M6032"*"

Reads a File Into Memory to be started with M6045

M6040 I100

Reboot I defines the delay before restart

MM6045 I4000 "M24"

Waits delay time and prints the M6032 open file on memory

M7506 I636264 T0//0; 1; color_flip

Changes UI colours Negative/Positive/or switch watever its on

M8512 "configFile.gcode"

Dumps the current config.gcode EEPROM data into a file

M9005 "Network","Password"
